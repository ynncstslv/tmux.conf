# .tmux.conf

This `.tmux.conf` file is a really simple configuration for my TMUX. It just sets a new `<leader>`, binds some keys for vim navigation and a theme for the status bar with some plugins. It is designed to make the use of TMUX more user-friendly, efficient, and ergonomic.

## Usage

```bash
sudo apt install tmux
git clone https://github.com/tmux-plugins/tpm ~/.tmux/plugins/tpm
nvim ~/.tmux.conf
```
